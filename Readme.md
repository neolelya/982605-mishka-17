Mishka is a handmade knitwear online store

## Used technologies: 
* HTML
* CSS (used flex-boxes for page grid)
* Sass
* JS
* Build process (Gulp)
* BEM methodology 
* Retina support
* Responsive design

## Cross-browser compatibility
* Chrome 
* Firefox 
* Safari 
* Edge 
* Internet Explorer 11
